# Font Scalar

Font Scalar is a beginner-level JavaScript project that lets users change text size by clicking buttons.
It's a simple DOM practice project where you can switch the font size between Small, Medium, Large, and Huge.

# Features

Change text size with button clicks

Smooth font-size transition

Active-button highlight

# What I used

HTML

CSS

JavaScript (DOM events, forEach, switch)

# How to run

Just open the index.html file in a browser and click the buttons to scale the text.

Note: This is a basic practice project to build core JavaScript skills and DOM handling confidence.
